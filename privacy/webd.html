수집하는 데이터

WEbd는 어떠한 개인정보나 사용 데이터도 수집, 저장, 전송하지 않습니다.

데이터 저장 위치

사용자가 만든 모든 변경사항(스타일 규칙, 콘텐츠 편집, 색상 팔레트, 언어 설정)은 사용자의 브라우저 안에만(chrome.storage.local) 저장됩니다. 이 데이터는 외부 서버로 전송되지 않으며, 개발자를 포함한 누구도 접근할 수 없습니다.

권한 사용 목적

  • 사이트 접근(모든 URL): 사용자가 방문 중인 페이지에 편집 도구와 저장된 스타일을 표시·적용하기 위해서만 사용됩니다.
  • storage / unlimitedStorage: 변경사항(교체 이미지 포함)을 사용자의 브라우저에 저장하기 위해 사용됩니다.
  • scripting: 사용자가 요청할 때 페이지의 애니메이션/모션을 일시정지하기 위해 사용됩니다.

제3자 제공

제3자에게 제공하는 데이터가 없습니다. 광고, 분석 도구, 추적 기술을 사용하지 않습니다.

WEbd Privacy Policy

Last updated: July 16, 2026

Data Collection

WEbd does not collect, store, or transmit any personal information or usage data.

Where Data Is Stored

All changes you make (style rules, content edits, color palettes, language preference) are stored only in your browser (chrome.storage.local). This data is never sent to any external server and cannot be accessed by anyone, including the developer.

Permission Usage

  • Site access (all URLs): used solely to display the editing tools and apply your saved styles on pages you visit.
  • storage / unlimitedStorage: used to save your changes (including replaced images) in your browser.
  • scripting: used to pause page animations/motion when you request it.

Third Parties

No data is shared with third parties. WEbd uses no ads, analytics, or tracking technologies.
